# Sales Intelligent Dashboard

An interactive dashboard for analyzing sales performance and forecasting future revenue.

## Overview

This project provides a simple but powerful sales analytics dashboard built with Python.
It allows users to explore sales performance across products, regions, and time, while also generating a basic revenue forecast using a machine learning model.

The dashboard is built with Streamlit and provides an easy-to-use interface for exploring the data.

## Features

* Global revenue KPI
* Revenue by product
* Revenue by region
* Revenue by month
* Interactive data visualization
* Sales forecasting using a machine learning model

## Tech Stack

* Python
* Pandas
* Streamlit
* Scikit-learn

## Installation

Clone the repository:

git clone https://github.com/diogompintoo/SalesIntelegentDashboard.git

Install dependencies:

pip install -r requirements.txt

Run the dashboard:

streamlit run dashboard/app.py
